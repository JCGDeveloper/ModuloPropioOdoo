# Gestión de Nóminas

Módulo académico para Odoo que permite:

- Registrar nóminas con sueldo base, bonificaciones, deducciones, estado y justificante PDF.
- Calcular automáticamente la base imponible, el IRPF pagado y el sueldo neto.
- Controlar las bonificaciones/deducciones mediante líneas específicas con validaciones de importe.
- Generar declaraciones de renta anuales con un máximo de 14 nóminas por empleado y año.
- Mostrar menús y vistas específicas dentro de Recursos Humanos para facilitar el acceso.

Requisitos: depende de `base` y `hr`. Instala el módulo y accede al menú **RRHH → Nóminas** para empezar.
